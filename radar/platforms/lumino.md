AI-based SRE Observability platform leveraging AI/ML to process and analyze
telemetry data (logs, metrics, traces). Built on Agentic MCP client/server
architecture with dedicated Konflux SRE assistant for reasoning behind
real-time telemetry data from OpenShift and Konflux components.

Working with IBM ITBench to create OpenShift/Konflux scenarios for evaluation.
Implementing agentic behavior and historical records using Kubearchive for
PipelineRuns and TaskRuns. Connectivity to LogAn project merged. Addresses
challenges of inefficient data traversal, high storage costs, and slow root
cause analysis.
