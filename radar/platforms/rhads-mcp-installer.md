The tssc CLI now supports Model Context Protocol (MCP) as a first-class
citizen, providing an AI-orchestrated, guided, and self-service installer
for TSSC/RHADS platforms. Offers intuitive conversational interface for
complex deployments and ongoing management.

AI-powered installer presented to the BU. Planning to extract a reusable
framework from the RHADS installer for other Red Hat-based platforms.
Transforms traditionally challenging platform installation into a
"vibe platform engineering" experience.
