AI agent platform for Atlassian's Jira and Confluence ecosystem. Enables teams
to build custom AI agents that can interact with Jira issues, search across
workspaces, and automate workflows.

Currently setting up Rovo agent for Atlassian Cloud UAT environment. Cannot be
fully deployed until Red Hat's Jira cloud migration is complete. Once live, will
enable AI-powered assistance for issue triage, status updates, and cross-project
insights.
