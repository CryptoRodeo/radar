Agent that designs and develops E2E/system tests based on multiple sources
of data including customer support tickets, bugs, RFEs, and documentation.
Goal is to increase product quality by feeding the agent with the right
context per product.

Continuing iterations with CNV (varying ticket scopes). MTV focus shifted
to feature test coverage due to inadequate current customer case coverage.
Addresses the challenge of improving test coverage based on real customer
issues.
