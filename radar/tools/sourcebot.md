Open-source self-hosted code search tool with MCP server that enables
Red Hat flavored code assistant replies in IDEs like Cursor. For example,
generates pre-commit config files with infosec gitleaks without needing
to search and study documentation.

Upgraded to 4.8.1 with performance improvements. Working on research agent
for MCP integration to simplify IDE integration rules. Available at
https://sourcebot.dno.corp.redhat.com/
