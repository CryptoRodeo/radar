Using GenAI (agentic w/ MCP) to enhance documentation and allow documentation
updates to be an integral part of CI workflow. Merging changes automatically
followed by suggestions/PRs to documentation repos.

Working on this project with partners in Konflux (Eyal Edri & Shay Vatarescu
teams). Aims to keep documentation in sync with code changes automatically,
reducing documentation drift.
